plant leaf segmentation and classification

Project Description
This project proposes an end-to-end framework for plant leaf segmentation, morphological processing, and classification using techniques from Digital Image Processing (DIP) and **Random Forest Classifier.

Thresholding: Comparative analysis of Otsu, Global and Adaptive approaches for light invariant segmentation.  Morphology: Noise reduction and mask cleaning through Erosion, Dilation, Opening and Closing.  Evaluation: IoU / Jaccard Index computation from ground truth.  Classification: Features extraction then Random Forest classification.
